BerryBox client app
------------
BerryBox ads client app with ElectronJS.

Built with
---------------
- ElectronJS

Requirements
---------------
- NPM ^v6.9.0
- Node ^v10
- Electron CLI v11.1.1

Development
---------------
```bash
$ git clone https://github.com/beryb/client.git
$ cd client
$ npm install
$ npm start
```
